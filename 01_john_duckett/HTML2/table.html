TABLES

What is a table?

A table is a representation of information in grid format. In HTML a table is written out row by row.

Below is an example of a simple table


10 20 30 40
5 10 15 20
50 200 450 800

Now let's add the Table heading element.

The Table Heading element is used to represent the table heading for either a row or a column

Let's take an example


Saturday Sunday
Tickets sold 13 20
Total sales #50,000 #75,000

Spanning Columns

You might want a cell to run across more than one column. This can be achieved using the "colspan" attribute then specifying the number of colums you want the cell to run through. It can be used alongside a "th" or "td" element.

Let's take an example.


9am 10am 11am 12am
Monday Maths Eng Bio
Tuesday Chem Gym

Spanning Rows

You might also want a cell to run across more than one row. This can be achieved using the "rowspan" attribute then specifying the number of rows you want the cell to run down the table. It can be used alongside a "th" or "td" element.

Let's take an example.


AIT NTA LTV
6-8pm Music Cartoon Comedy
8-10 movie
